# Security

**Description:**

This command will open a panel **(Admin Only)** which allows you to enable or disable the **Security & Safety** feature.

#### Usage:

```
<prefix>Security
```

#### Permissions:

```
`administrator` premissions
```

#### Arguments:

```
None
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
