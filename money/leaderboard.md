# Leaderboard

**Description:**

Shows you the leaderboard

#### Usage:

```
<prefix>leaderboard
<prefix>lb
```

#### Permissions:

```
`Manage Channels` permission
```

#### Arguments:

```
{channel} - A channel mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
