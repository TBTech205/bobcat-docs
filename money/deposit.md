# Deposit

#### **Description:**

Deposit money into your bank account

#### Usage:

```
<prefix>deposit [amount]
<prefix>dp [amount]
```

#### Permissions:

```
None
```

#### Arguments:

```
[amount] - Amount you want to deposits into your bank
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
