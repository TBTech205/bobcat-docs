# Send



#### **Description:**

Sends money to members

#### Usage:

```
<prefix>send [member] [amount]
```

#### Permissions:

```
None
```

#### Arguments:

```
[member] - A member mention or ID
[amount] - Amout you want to send
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
