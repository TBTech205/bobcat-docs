# Buy

**Description:**

Buy things from the shop

#### Usage:

```
<prefix>buy [item] {amount}
```

#### Permissions:

```
None
```

#### Arguments:

```
[item] - A item from the shop
{amount} - How many you want to buy
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
