# Withdraw

#### **Description:**

Withdraws money from your bank

#### Usage:

```
<prefix>withdraw [amount]
<prefix>wd [amount]
```

#### Permissions:

```
None
```

#### Arguments:

```
[amount] - Amount you want to withdraw from your bank
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
