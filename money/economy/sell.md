# Sell

**Description:**

Sells your items

#### Usage:

```
<prefix>sell [item] {amount}
```

#### Permissions:

```
None
```

#### Arguments:

```
[item] - A item from the shop
{amount} - How many you want to sell
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
