# Giverole

## ⚠️ Disabled&#x20;



**Description**

Gives a role to a member

### Usage

```
b!giverole [user] [role]
b!addrole [user] [role]
```

### Permissions

```
`Manage Roles` permissions
```

### Arguments

```
[user] - A member mention or ID
[role] - A role mention or ID
```

#### Syntax

```
[] - Required.
{} - Optional.
```
