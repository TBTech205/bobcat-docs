# Emoji

**Description:**

Creates a emoji

#### Usage:

```
<prefix>createemoji [url] [name]
<prefix>steal [url] [name]
<prefix>ae [url] [name]
<prefix>ce [url] [name]
```

#### Permissions:

```
`Manage Emojis` permissions
```

#### Arguments:

```
[url] - Emoji url
{name} - A name for the emoji
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
