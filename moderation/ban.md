# Ban

**Description:**

Bans the member you mentioned

#### Usage:

```
<prefix>ban [member] {reason}
```

#### Permissions:

```
`Ban Members` permission
```

#### Arguments:

```
[member] - A member mention or User ID
{reason} - A reason for action
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
