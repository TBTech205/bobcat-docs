# Mute

**Description:**

Mutes a member from the server

#### Usage:

```
<prefix>kick [member] {reason}
```

#### Permissions:

```
`Kick Members` permissions
```

#### Arguments:

```
[member] - A member mention or User ID
{reason} - A reason for action
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
