# Giveaway

**Description:**

Creates a giveaway

#### Usage:

```
<prefix>giveaway [mins] [prize]
```

#### Permissions:

```
`Manage Channels` permissions
```

#### Arguments:

```
[mins] - Sets the duration
[prize] - Sets the prize
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
