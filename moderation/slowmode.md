# Slowmode

**Description:**

Sets the channels slow**-**mode

#### Usage:

```
<prefix>slowmode [seconds]
```

#### Permissions:

```
`Manage Channels` permission
```

#### Arguments:

```
[seconds] - time in SECONDS
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
