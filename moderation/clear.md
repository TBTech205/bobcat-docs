# Clear

**Description:**

Deletes a number of messages in a channel

#### Usage:

```
<prefix>clear [amount]
<prefix>purge [amount]
```

#### Permissions:

```
`Manage Messages` permission
```

#### Arguments:

```
{amount} - The amount of messages you want to delete
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
