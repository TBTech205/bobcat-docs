# Lockdown

**Description:**

Locks** **down the channel

#### Usage:

```
<prefix>lockdown {channel}
```

#### Permissions:

```
`Manage Channels` permission
```

#### Arguments:

```
{channel} - A channel mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
