# Unmute

**Description:**

Unmutes a member from the server

#### Usage:

```
<prefix>unmute [member]
```

#### Permissions:

```
`Kick Members` permissions
```

#### Arguments:

```
[member] - A member mention or User ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
