# Takerole

## ⚠️ Disabled



**Description:**

Removes a role to a member

#### Usage:

```
b!removerole [user] [role]
b!takerole [user] [role]
```

#### Permissions:

```
`Manage Roles` permissions
```

#### Arguments:

```
[user] - A member mention or ID
[role] - A role mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
