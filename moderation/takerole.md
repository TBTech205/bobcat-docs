# Takerole

**Description:**

Removes a role to a member

#### Usage:

```
<prefix>removerole [user] [role]
<prefix>takerole [user] [role]
```

#### Permissions:

```
`Manage Roles` permission
```

#### Arguments:

```
[user] - A member mention or ID
[role] - A role mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
