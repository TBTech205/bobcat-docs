# Kick

**Description:**

Kicks a member from the server

#### Usage:

```
<prefix>kick [member] {reason}
```

#### Permissions:

```
`Kick Members` permission
```

#### Arguments:

```
{Mention} - A user mention or User ID.
[Reason] - A reason for kick.
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
