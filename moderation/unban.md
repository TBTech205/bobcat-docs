# Unban

**Description:**

Unbans a member from the server

#### Usage:

```
<prefix>unban [member]
```

#### Permissions:

```
`Ban Member` permission
```

#### Arguments:

```
[member] - A member mention or User ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
