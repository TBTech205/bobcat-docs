# Levelling

**Description**

This command will open a panel **(Admin Only)** which allows you to enable or disable the **Levelling system** feature.

### Usage

```
b!help > Settings > Levelling
```

### Permissions

```
administrator permissions
```

### Arguments

```
None
```

### Syntax

```
[] - Required.
{} - Optional.
```
