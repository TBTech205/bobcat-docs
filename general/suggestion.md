# Suggestion

**Description:**

Sends your command suggestion

#### Usage:

```
<prefix>suggest [trigger] [response]
<prefix>suggestion [trigger] [response]
```

#### Permissions:

```
None
```

#### Arguments:

```
[trigger] - name of the command e.g. help
[response] - what will it do
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
