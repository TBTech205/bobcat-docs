# 8-Ball

**Description:**

This plays a magic 8 ball

#### Usage:

```
<prefix>8ball [question]
<prefix>ask [question]
```

#### Permissions:

```
No permissions needed for this command
```

#### Arguments:

```
[question] - input your question
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
