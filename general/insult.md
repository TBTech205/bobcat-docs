# Insult

**Description:**

Insults a yourself or a member (this can be harsh)

#### Usage:

```
<prefix>insult {member}
```

#### Permissions:

```
No permissions needed for this command
```

#### Arguments:

```
{member} - Mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
