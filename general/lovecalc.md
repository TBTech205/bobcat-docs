# Lovecalc

**Description:**

calculates your love bettween 2 people

#### Usage:

```
<prefix>lovecalc [member1] [member2]
```

#### Permissions:

```
No permissions needed for this command
```

#### Arguments:

```
[member1] - Mention or ID
[member2] - Mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
