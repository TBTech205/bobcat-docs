# Howhot

**Description:**

Calculates how hot you are

#### Usage:

```
<prefix>howhot [member]
<prefix>hotcalc [member]
<prefix>hot [member]
```

#### Permissions:

```
No permissions needed for this command
```

#### Arguments:

```
[member] - Mention or ID
```

#### Syntax:

```
<prefix> - b! or custom prefix
[] - Required.
{} - Optional.
```
