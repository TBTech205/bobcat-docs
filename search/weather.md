# Weather

### **Description**

Search for the weather on [openweathermap.org](https://openweathermap.org/)

### Usage

```
b!weather [city] {unit}
```

### Permissions

```
None
```

### Arguments

```
[city] - Location you want to look up
{unit} - f or c (default: f)
```

### Syntax

```
[] - Required.
{} - Optional.
```
